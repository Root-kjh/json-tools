## Extension Name
JSON Tools - Format, Validate & Convert

## Short Description (132 chars max)
Privacy-first JSON utilities. Format, validate, minify, and convert JSON instantly. All processing happens locally in your browser.

## Detailed Description

JSON Tools is a powerful, privacy-first extension for developers. All processing happens locally in your browser - your data never leaves your device.

FEATURES:

Format JSON
- Beautify messy JSON with customizable indentation (2 spaces, 4 spaces, or tabs)
- Instantly see your formatted output

Validate JSON
- Check if your JSON is valid
- Get detailed error messages for invalid JSON
- See statistics about your JSON structure

Minify JSON
- Compress JSON by removing whitespace
- See compression ratio and byte savings

Convert JSON
- JSON to TypeScript interfaces
- JSON to YAML
- JSON to CSV
- JSON to XML

MORE FEATURES:
- Context menu: Right-click on selected text to format, minify, or validate
- Keyboard shortcut: Cmd+Shift+J (Mac) or Ctrl+Shift+J (Windows)
- Paste from clipboard with one click
- Copy output to clipboard
- Dark theme optimized for developers

PRIVACY:
- 100% local processing
- No data sent to servers
- No analytics or tracking
- No account required

Perfect for:
- API developers debugging responses
- Frontend developers working with JSON data
- Backend developers validating configurations
- Anyone who works with JSON regularly

Open source: https://github.com/Root-kjh/json-tools
