# JSON Tools

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Free, privacy-first online JSON utilities. All processing happens in your browser - your data never leaves your device.

**Live Demo:** [https://json.jobby-time.com](https://json.jobby-time.com)

## Features

- **JSON Formatter & Validator** - Format, beautify, minify, and validate JSON with customizable indentation
- **JSON to TypeScript** - Generate TypeScript interfaces from JSON data
- **JSON to CSV** - Convert JSON arrays to CSV format for spreadsheets
- **JSON Diff** - Compare two JSON objects and visualize differences
- **JSON Path Finder** - Click on any value to get its JSONPath

## Why JSON Tools?

| Feature | Benefit |
|---------|---------|
| **Privacy First** | All processing happens client-side. Your JSON data never leaves your browser. |
| **Lightning Fast** | No server round-trips. Instant results powered by your browser. |
| **All-in-One** | Format, validate, convert, compare - all your JSON needs in one place. |
| **No Ads** | Clean, distraction-free interface. |

## Tech Stack

- React 19
- TypeScript
- Vite 7
- Tailwind CSS

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Lint
npm run lint
```
